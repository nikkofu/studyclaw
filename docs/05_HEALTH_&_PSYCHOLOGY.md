# StudyClaw 深度智能化与儿童健康心理辅导机制 (Health & Psychology)

为了体现 StudyClaw 作为家庭级 Agentic 生态的独特性，我们将儿童的**生理健康**（用眼、坐姿）与**心理健康**（情绪疏导、正向激励）纳入到产品设计的核心考量中。AI 既是严师也是益友。

## 1. 生理健康守护：环境与体态感知

### 1.1 动态用眼健康管理
相比于传统的“每 30 分钟锁定屏幕”死板设定，Agentic 守护应当更平滑且具备商榷性：
- **光线感知**: 读取 Pad 的环境光传感器。当环境过暗时，Agent 主动提示：“环境太暗啦，小主人的眼睛会累的，我们去开个灯再继续好吗？”如果 1 分钟后未改善，暂时调低对比度并中止新任务布置。
- **距离与坐姿检测 (Vision / 陀螺仪)**: 依靠前置摄像头估计人脸距离，或利用陀螺仪判断 Pad 是否被平躺举着看。如果距离屏幕过近（< 30cm），画面四角会出现“柔和的毛玻璃遮罩”效果，伴随 Agent 语音建议：“坐直一点哦，像我一样把脊背挺直。”
- **柔性番茄钟休憩**: 当识别到孩子已经看屏幕连续解题专注超过 20 分钟，Agent 会在当前节点（刚好做完一题的间隙）切入：“哇，你太专注了！来，跟我一起看这片远处的森林，我们做个深呼吸……”（屏幕展示 20 秒放松护眼动画，之后自动返回）。

## 2. 心理健康辅导：从纠错到共情

### 2.1 挫折应对与抗压能力构建
传统的 APP 在做错题时仅给出大大的红叉或 `False`，这对 6-12 岁儿童的心理暗示是负面的。
- **成长型思维话术引导 (Growth Mindset)**:
  - **错误反馈**: 当判定拼写或算术错误时，Agent 触发话术库中的“鼓励分类”。例如：“这道题确实很有挑战性呢！我们来看看是哪一步卡住了，再试一次你一定能行！”
  - **降低焦虑**: 对于反复出错的题目，Agent 主动介入熔断机制：“我看这道题咱们想了很久了。没关系，有时候换换脑子会有新灵感，我们先复习一下前天做过的这几道简单的，找找感觉，晚点再来挑战大魔王！”
- **“进步”比“完美”更重要**: 奖励系统设计应当倾斜于“努力度”而非“绝对正确率”。即便全错，只要孩子在屏幕前坚持听完了讲解并跟着读，Agent 也会给予“勇敢探索”徽章。

### 2.2 情绪日志与非正式倾听 (Virtual Couch)
儿童往往很难向严厉的家长袒露心扉，此时 Agent 可以充当“倾听树洞”。
- **情绪签到 (Daily Mood Check-in)**: 每天打开 App 或者睡前日记时，界面上会有几个可爱的小动物表情（开心、平静、生气、委屈）。孩子点击相应的表情。
- **引导性发问**: 如果点击“委屈”，Agent 会用极温柔的声音问：“今天遇到什么不开心的事情了吗？可以悄悄告诉我，我会替你保密的哦。”（此时录音上传至云端供家长后台查看摘要，但界面的确给了孩子抒发出口）。
- **心理压力监测 (Parent Insights)**: Agent 引擎后台提取近期日志的 Sentiment Analysis (情感极性)。如果系统察觉到孩子连续多天情绪低落或在特定科目（如英语）出现高频焦虑词汇，会向家长端的微信小程序发送《心理波动微报告》，并附上专家建议：“孩子近期对英语听写表现出较高抵触感，建议降低本周词汇量，转为趣味阅读。”

## 3. 游戏化与正强化心理学应用

- **斯金纳箱的随机奖励**: 除了固定的积分（例：读完一首诗 = 5 Claw 币），Agent 每次在阶段性任务完成时，有 10% 概率触发“神秘宝箱”（如 3 倍爆率或特殊主题皮肤），激发多巴胺，提升翌日继续学习的内驱力。
- **去竞争化的个体成长轴**: 系统**默认不设公开排行榜**（或只提供匿名时长比拼），避免带给孩子同辈压力（Peer Pressure）。UI 视觉中心永远是孩子自己与过去的对比：“你今天比上周一多专注了 15 分钟呢！你的小飞船已经飞出了太阳系！”

## 4. 技术与伦理约束
心理与健康辅导必须坚守伦理底线。
1. **绝不允许 Agent 诊断疾病**: 任何涉及到疑似多动症（ADHD）或抑郁倾向的极端词汇（如“我想死”），Agent 必须立刻触发最高等级的安全守卫（Safety Guardrail）——**终止闲聊**，回复标准化安抚话术，并**立即产生强提醒给家长**（如短信或电话接警 API）。
2. **避免路径依赖**: Agent 的目的终极是让孩子能够离开 Agent 也能独立生活与学习。因此系统要有“防过度沉迷机制”，周末应当主动鼓励孩子：“今天天气真好，去院子里跑两圈或者陪爸爸妈妈去楼下散步吧，不需要一直陪着我啦~”
