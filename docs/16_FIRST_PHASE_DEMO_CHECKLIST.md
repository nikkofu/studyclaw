# StudyClaw 第一阶段演示清单

本文档用于第一阶段产品能力演示、联调验收和发布前功能走查。

适用目标：

- 版本目标：`v0.2.0`
- 目标日期：`2026-03-10`

推荐先完成基础检查：

1. `bash scripts/check_no_tracked_runtime_env.sh`
2. `bash scripts/preflight_local_env.sh`
3. 启动 Go 后端
4. `bash scripts/smoke_local_stack.sh`
5. `bash scripts/demo_local_stack.sh`

如果本次目标不是“演示”，而是“第一阶段正式签收前复核”，还应先阅读：

- `docs/17_DELIVERY_READINESS.md`

## 1. 演示前准备

### 1.1 启动服务

后端：

```bash
cd apps/api-server
go run ./cmd/studyclaw-server
```

家长端：

```bash
cd apps/parent-web
npm run dev -- --host 0.0.0.0
```

Pad 端：

```bash
cd apps/pad-app
flutter run --dart-define=API_BASE_URL=http://localhost:8080 -d chrome
```

### 1.2 建议演示用固定日期

建议统一使用固定的 `assigned_date`，避免“今天 / 明天”混淆：

- `2026-03-10`

### 1.3 建议演示用固定数据

建议固定：

- `family_id=306`
- `assignee_id=1`

## 2. 第一阶段演示主线

下面 7 项就是第一阶段目标功能演示清单。

### 2.1 家长发布当天作业

目标：

- 演示家长如何把老师原始作业布置到某一天

演示动作：

1. 打开家长端
2. 选择孩子
3. 选择日期 `2026-03-10`
4. 粘贴学校群风格原文
5. 点击解析

通过标准：

- 家长端成功进入解析草稿视图

### 2.2 AI 解析并生成任务草稿

目标：

- 演示系统按学科 / 分组 / 原子任务拆解原文

演示动作：

1. 查看解析结果
2. 确认能看到：
   - `subject`
   - `group_title`
   - `title`
   - `confidence`
   - `needs_review`
3. 如有高风险任务，演示人工审核
4. 确认发布

通过标准：

- 任务成功发布到指定日期
- 高风险任务不会被静默忽略

### 2.3 Pad 打开当天任务并逐项完成

目标：

- 演示孩子端如何完成当天任务

演示动作：

1. 切到 Pad
2. 打开 `2026-03-10` 任务板
3. 勾选一条单任务
4. 勾选一个分组
5. 如支持，演示全部完成

通过标准：

- 勾选动作成功同步
- 页面有明确反馈

### 2.4 家长查看当日统计与同步结果

目标：

- 演示家长端能看到当天完成情况

演示动作：

1. 回到家长端
2. 刷新当天页面
3. 查看：
   - 总任务数
   - 已完成数
   - 未完成数
   - 学科完成情况
   - 最近同步时间

通过标准：

- 家长端能看到 Pad 端刚刚完成的变化

### 2.5 单词清单逐词播放

目标：

- 演示家长给清单，Pad 逐词播放

演示动作：

1. 家长端创建一份中文或英语单词清单
2. 绑定到孩子和 `2026-03-10`
3. 切到 Pad
4. 进入单词播放模式
5. 播放当前词
6. 点击“下一个”
7. 演示“重播当前词”

通过标准：

- Pad 端能按顺序逐词播放
- 能展示播放进度

### 2.6 积分变化

目标：

- 演示自动积分和手工积分

演示动作：

1. 在任务完成后观察自动积分变化
2. 家长端手工执行一次加分或扣分
3. 输入原因：
   - 表扬
   - 批评
   - 作业完成奖励
4. 切到孩子端查看积分展示

通过标准：

- 家长端和孩子端都能看到当前积分结果
- 积分变化有可解释来源

### 2.7 日 / 周 / 月数据与 AI 鼓励

目标：

- 演示图表和正向反馈

演示动作：

1. 家长端打开日视图
2. 家长端打开周视图
3. 家长端打开月视图
4. 查看 AI 输出的鼓励性总结
5. 如孩子端有简化版入口，也同步演示

通过标准：

- 图表可见
- AI 总结内容和统计结果一致
- 文案风格积极、支持型

## 3. 建议演示顺序

推荐按下面顺序走：

1. 家长发布作业
2. AI 解析与审核
3. Pad 完成任务
4. 家长查看统计
5. 单词播放
6. 积分变化
7. 日 / 周 / 月反馈
