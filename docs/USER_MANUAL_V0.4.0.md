# StudyClaw v0.4.0 使用说明

发布日期：`2026-03-16`

## 这份手册的定位

`v0.4.0` 是第二阶段 `Agentic 学习助手` 的正式版本。

这版对家长最重要的变化是：

1. 发布背诵 / 朗读任务时，就可以先把学习素材准备好
2. 孩子在 Pad 上完成一次语音学习后，家长端可以直接看到结果摘要
3. 家长可以根据结果决定是否让孩子再练一次

## 给家长

### 1. 布置背诵 / 朗读任务时先做什么

推荐顺序：

1. 打开家长端 `发布`
2. 选择孩子和日期
3. 粘贴老师原文或群消息
4. 在审核卡里确认：
   - 标题
   - 作者 / 来源
   - 标准原文
   - 是否对孩子隐藏正文
5. 再确认发布

### 2. 孩子完成语音学习后去哪里看

打开家长端 `反馈`，再切到 `语音`。

这里会看到：

- 本次语音学习摘要
- 标题 / 作者识别结果
- 完成度
- 逐句不稳的重点
- 孩子真实开口记录
- 是否建议重练

### 3. 家长怎么判断要不要重练

如果页面显示：

- `建议重练`
- 某些句子 `部分对上` 或 `差距较大`
- 系统建议“把第一句再熟读一遍后重背”之类提示

那就可以让孩子重点再练这些句子，而不是整篇盲目重来。

## 给孩子

### 1. 开始语音学习时怎么做

在 Pad 的语音工作台里：

1. 点击 `开始说话`
2. 正常背诵或朗读
3. 中途停几秒没关系
4. 结束时再点 `结束说话`

### 2. 为什么不会因为停顿马上结束

因为 `v0.4.0` 已经把主流程改成“人工开始到人工结束”。

这表示：

- 中途短暂停顿不会立刻结束
- 你可以按自己的节奏继续说
- 系统会把真实停顿整理成学习记录

### 3. 完成后会发生什么

系统会：

- 整理 transcript 分段
- 对照标准原文分析
- 给出鼓励
- 把结果同步给家长端复盘

## 给演示和测试同事

### 1. 最短回归路径

1. 家长端发布一个背诵或朗读任务，并确认参考原文
2. Pad 端进入学习语音工作台
3. 点击 `开始说话`
4. 中途停顿后继续说
5. 点击 `结束说话`
6. 确认 Pad 出现 transcript 分段与分析结果
7. 回到家长端 `反馈 -> 语音`
8. 确认家长端出现：
   - 语音学习摘要
   - 完成度
   - 逐句对照重点
   - 重练建议

## 一句话结论

`v0.4.0` 的核心不是让孩子“多一个语音按钮”，而是让家长第一次真正能看到一次语音学习的结果，并据此复盘和干预。
